# 🏦 Loan Eligibility Prediction System

This is a full-stack Machine Learning web application that predicts whether a loan applicant is eligible for a loan based on various financial and demographic factors.

## 🛠️ Features
- **Accurate Predictions:** Uses a Random Forest Classifier trained on historical housing finance data.
- **Interactive UI:** Clean and responsive web interface built with Flask and HTML/CSS.
- **Instant Result:** Real-time feedback on loan approval status (Approved/Rejected).

## 🧰 Tech Stack
- **Languages:** Python, HTML5, CSS3
- **Machine Learning:** Scikit-learn, Pandas, NumPy
- **Backend:** Flask
- **Server:** Gunicorn
- **Deployment:** Render

## 📊 Data Overview
The model is trained on factors such as:
- Credit History (Most critical factor)
- Applicant Income & Co-applicant Income
- Loan Amount & Term
- Education, Marriage Status, and Dependents
